# 📊 Mobile Sales Data Analysis Report

## 📌 Project Overview
The **Mobile Sales Dashboard** is a data visualization project developed using **Microsoft Power BI** to analyze and monitor mobile phone sales performance.  
This interactive dashboard helps businesses track important sales metrics, identify trends, and make data-driven decisions.

The dashboard provides insights into **sales performance, product demand, customer transactions, and payment methods**, allowing stakeholders to better understand market behavior and improve sales strategies.

---

## 🎯 Project Objectives
The main objectives of this project are:

- Analyze overall **mobile phone sales performance**
- Monitor **Month-to-Date (MTD) sales trends**
- Compare **Year-over-Year (YoY) sales performance**
- Identify **top-performing mobile models**
- Understand **customer transaction behavior**
- Analyze **payment method distribution**
- Provide clear **data-driven insights for business decisions**

---

## 📊 Dashboard Features

### 1️⃣ Key Performance Indicators (KPIs)
The dashboard highlights important sales metrics including:

- **Total Sales:** 59M
- **Total Quantity Sold:** 1K
- **Total Transactions:** 291
- **Average Price:** 40.98K

These KPIs provide a quick overview of the business performance.

---

### 2️⃣ Sales Trend Analysis
- Monthly sales quantity trends
- Identification of **peak and low sales months**
- Helps understand **seasonal demand patterns**

---

### 3️⃣ Customer Rating Analysis
Customer feedback is categorized into:

- Good Ratings
- Average Ratings
- Poor Ratings

This helps evaluate **customer satisfaction and product quality perception**.

---

### 4️⃣ Payment Method Analysis
The dashboard analyzes transactions by payment methods such as:

- Cash
- UPI
- Credit Card
- Debit Card

This insight helps businesses understand **preferred payment methods among customers**.

---

### 5️⃣ Mobile Model Performance
The dashboard compares **total sales by mobile model**, helping identify:

- Best-selling devices
- Product demand trends
- Inventory planning insights

Examples:
- Galaxy Note 20
- Vivo Y51
- OnePlus Nord

---

###6️⃣ Sales by Day Analysis
The dashboard analyzes sales by **day of the week**, helping identify:

- Highest sales days
- Customer purchase patterns
- Optimal promotional timing
