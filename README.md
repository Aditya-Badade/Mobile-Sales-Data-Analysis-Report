# Mobile-Sales-Data-Analysis-Report
The Mobile Sales Dashboard is a data visualization project developed using Power BI to analyze and monitor mobile phone sales performance. The dashboard provides interactive insights into sales trends, customer transactions, product performance, and payment methods.
Key Features
Interactive Power BI Dashboard
Analysis of Total Sales, Quantity Sold, Transactions, and Average Price
Month-to-Date (MTD) Sales Tracking
Year-over-Year (YoY) Sales Comparison
Sales analysis by Mobile Model
